# E-Learning Website  
A responsive multi-page e-learning website built using HTML, CSS, Bootstrap, and JavaScript.  
This project showcases a modern front-end design with structured navigation, course pages, instructor details, contact forms, and interactive components.

---

## Features

### Website Pages
- Home page  
- Courses page  
- Single course details page  
- Instructor / Team page  
- About page  
- Contact page  
- Login and Signup pages  
- Testimonials page  

### Front-End Features
- Fully responsive layout using Bootstrap grid system  
- Clean UI components and reusable sections  
- Navigation bar and footer across all pages  
- Cards for course listings  
- Form validation using JavaScript  
- Organized folder structure for maintainability  

---

## Technologies Used

### Frontend
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  

### Assets and Styling
- Custom SCSS files  
- Reusable CSS components  
- Image assets organized in dedicated folders
